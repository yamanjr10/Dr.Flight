About DR.Flight

An arcade game written with C# and powered by Unity engine. The game is inspired by flappy bird and space arcade. It is titled as 'DR.Flight'. It consists of a space game object that will have to dodge and destroy the obstacles in order to survive and complete the objectives.

We have also added a score system which makes the game harder as it progresses. We hope to revive the arcade game industry that has been rather silent since the digitalized high graphics games took over the media.
